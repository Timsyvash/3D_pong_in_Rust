# 3D Pong / Breakout on Rust (Bevy)

Проста тривимірна аркадна гра (варіація на тему Понгу та Арканоїду), створена на мові програмування **Rust** за допомогою сучасного ECS-рушія **Bevy**.

---

## 🎮 Особливості гри

* **3D Простір:** Гра використовує тривимірні геометричні примітиви (куби, сфери) та динамічне освітлення з тінями.
* **Просунута фізика м'яча:** Напрямок та кут польоту м'яча залежать від того, якою саме частиною ракетки (ближче до центру чи до краю) ви його відбили.
* **Система станів (Game States):** Реалізовано повноцінний ігровий цикл: підрахунок очок, облік життів, екран програшу (Game Over) та можливість поставити гру на паузу.
* **Кастомний UI:** Графічний інтерфейс користувача для відображення поточної статистики в реальному часі.

---

## 🕹️ Керування

* **A** або **Стрілка вліво (←)** — Рух ракетки ліворуч
* **D** або **Стрілка вправо (→)** — Рух ракетки праворуч
* **P** — Поставити гру на паузу / Зняти з паузи
* **Пробіл (Space)** — Перезапустити гру (доступно після екрана Game Over)

---

## 🛠️ Встановлення та налаштування

Щоб запустити гру, у вас має бути встановлений інструментарій **Rust** (включаючи менеджер пакетів `cargo`).

### 📂 Важливо: Структура ассетів
Гра використовує кастомний шрифт для UI. Переконайтеся, що у корені вашого проєкту створено папку `assets` з такою структурою файлів:
